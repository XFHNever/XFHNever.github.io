数据库基本操作指令

  1. 查看命令提示： help

    db.help();
db.yourCollectionName.help();
db.yourCollectionName.find().help();
rs.help();

  2. 显示数据库列表：

    show dbs;

  3. 显示当前数据库中的集合（类似关系数据库中的表）: 

    show collections;

  1. 切换/创建数据库： 

    use yourDB;   当数据库已经存在时，将会切换到指定数据库，否则将会创建新的数据库。

  1. 删除当前使用的数据库：

    db.dropDatabase();

  2. 查看当前DB的状态： 

    db.stats();

  3. 查看当前使用的数据库： 

    db.getName()或者db;

  4. 当前db版本： 

    db.version();

  5. 查看当前db的链接机器地址:

    db.getMongo();

  6. 从指定主机上克隆数据库: 

    db.cloneDatabase(“127.0.0.1”); 将指定机器上的数据库的数据克隆到当前数据库.

  7. 从指定的机器上复制指定数据库数据到某个数据库: 

    db.copyDatabase("mydb", "temp", "127.0.0.1");将本机的mydb的数据复制到temp数据库中

  8. 修复当前数据库: 

    db.repairDatabase();

  9. 查看数据库服务器状态： 

    db.serverStatus();

  10. 查询指定数据库包含的集合名称列表:

    db.getCollectionNames();

用户相关操作

  1. 查看当前所有用户： 

    show users;

  2. 添加一个用户：

    db.addUser("userName"); 
db.addUser("userName", "password", true); 添加用户、设置密码、是否只读

  3. 删除用户： 

    db.removeUser("userName");

  4. 数据库认证、安全模式：

    db.auth("userName", "password");

数据集（表）操作

  1. 创建集合： 

    db.createCollection(“collName”);

  2. 得到指定名称的聚集集合: 

    db.getCollection("collName");

  3. 显示当前db所有聚集索引的状态:

    db.printCollectionStats();

  4. 查询当前集合的数据条数: 

    db.yourColl.count();

  5. 查看数据空间大小: 

    db.yourColl.dataSize();

  6. 得到当前聚集集合所在的db: 

    db.yourColl.getDB();

  7. 得到当前聚集的状态

    db.yourColl.stats();

  8. 得到聚集集合总大小 

    db.yourColl.totalSize();

  9. 聚集集合储存空间大小 

    db.yourColl.storageSize();

  10. Shard版本信息 

    db.yourColl.getShardVersion()

  11. 聚集集合重命名 

    db.yourColl.renameCollection("toColl"); 将yourColl重命名为toColl

  12. 删除当前聚集集合 

    db.yourColl.drop();

数据集查询语句

  1. 查询所有记录：

    db.yourColl.find();

  2. 查询去掉后的当前聚集集合中的某列的重复数据：

    db.yourColl.distinct("name");会过滤掉name中的相同数据

  3. 条件查询： 

    db.yourColl.find({查询条件},{指定键}); 

    - 查询age = 22的记录: db.yourColl.find({"age": 22});
    - 查询age >= 23 并且 age <= 26: db.yourColl.find({age: {$gte: 23, $lte: 26}});
    - 查询name中以mongo开头的: db.yourColl.find({name: /^mongo/});

比较符号：$lt,$lte,$gt,$gte,$ne分别对应>,>=,<,<=,!=

  4. 查询后排序： 

    db.yourColl.find().sort({age:1}); 
按照age升序，age：-1按照降序。

  5. 查询前十条数据： 

    db.yourColl.find().limit(10);

  6. 查询10条以后的数据： 

    db.yourColl.find().skip(10);

  7. 查询5到10之间的数据: 

    db.yourColl.find.limit(10).skip(5); 
可用于分页，limit是pageSize，skip是第几页*pageSize

  8. 查询第一条数据： 

    db.yourColl.findOne();

  9. 包含于不包含： 

    $in或$nin 
db.yourColl.find({age:{$in:[25,17]}});

  10. OR查询： 

    查询年龄大于20或者班级号小于5的学生 
   db.yourColl.find({$or:[{age:{$gt:20}},{grade:{$lt:5}}]})

  11. $not查询： 

    查询所有名字中不含有“li”的学生 
    db.yourColl.find({name:{$not:/li/i}});

  12. 数组查询$all和index：

    查询喜欢看MongoDB和JS的学生：
  db.yourColl.find({books:{$all:["MongoDB","JS"]}});
查询第二本书是java的学生：
  db.yourColl.find({"books.1":"java"}); 
【books是一个数组，也是yourColl中的一个键】

  13. 查询某个结果集的记录条数：

    db.yourColl.find({age: {$gte: 25}}).count();

  14. $slice查询返回指定数组的内部值： 

    查询jim书架中第2~4本书：
db.yourColl.find({name:"jim"},{books:{"$slice":[1,3]}}); 

最后一本的话{"$slice":-1}.

  15. 查询快照：

    db.yourColl.find({$query:{name:"jim"},$snapshot:true})

  16. Group查询：

    db.runCommand({group:{
    ns:"persons",              //集合名称
    key:{"country":true},      //分组的键对象
    initial:{m:0},             //初始化累加器
    $reduce:function(doc,prev){     //组分解器
        if(doc.m > prev.m){
            prev.m = doc.m;
            prev.name = doc.name;
            prev.country = doc.country;
        }
    },
    finalize:function(prev){      //组完成器
        prev.m = prev.name+" Math scores "+prev.m
    },
    condition:{m:{$gt:90}}       //条件
}})

数据集插入更新删除记录语句

  1. 插入：

    db.yourColl.insert({});

  2. 批量插入：

    shell不支持批量插入，可以用Mongo的应用驱动或是shell的for循环。

  3. save操作: 

    db.yourColl.save({});和insert的区别是：当遇到_id相同的情况下，save完成保存，insert则会报错。

  4. 删除列表中所有数据： 

    db.yourColl.remove();

  5. 根据条件删除: 

    db.yourColl.remove({}); 注：对于数据量十分庞大的集合，直接删除该集合并重建索引比直接用remove效率高。

  6. 更新： 

    db.yourColl.update({查询器},{修改器});

  7. insertOrUpdate操作： 

    db.yourColl.update({查询器},{修改器}，true);查询出就更新，否则插入.

  8. 批量更新： 

    db.yourColl.update({查询器},{修改器},false,true);

  9. 查询修改删除：

    db.yourColl.findAndModify({

    query: {age: {$gte: 25}}, 

    sort: {age: -1}, 

    update: {$set: {name: 'a2'}, $inc: {age: 2}},

    remove: true

});

db.runCommand({ findandmodify : "yourColl", 

    query: {age: {$gte: 25}}, 

    sort: {age: -1}, 

    update: {$set: {name: 'a2'}, $inc: {age: 2}},

    remove: true

});

索引

  1. 创建索引：

    db.yourColl.ensureIndex({name: 1});  
1是指正序；-1倒序。

  2. 查询当前聚集集合所有索引: 

    db.yourColl.getIndexes();

  3. 查看总索引记录大小: 

    db.yourColl.totalIndexSize();

  4. 读取当前集合的所有index信息: 

    db.yourColl.reIndex();

  5. 删除指定索引:

    db.yourColl.dropIndex("name_1");

  6. 删除所有索引索引:

    db.yourColl.dropIndexes();

  7. 创建索引并制定索引的名字： 

    db.yourColl.ensureIndex({age:-1},{name:"indexName"});

  8. 创建唯一索引： 

    db.yourColl.ensureIndex({age:-1},{unique:true});不能插入重复的数值。

  9. 剔除重复数据： 

    db.yourColl.ensureIndex({age:-1},{unique:true,dropDups:true});

  10. 强制使用指定的索引： 

    db.yourColl.find({查询器}).hint({age:1}); 
注：age必须是已经创建过的索引。

  11. 查看本次查询使用的索引和状态信息： 

    db.yourColl.find({..}).explain();

  12. 查看数据库已经建立的索引： 

    db.system.indexes.find(); db.system.namespaces.find();

  13. 后台执行创建索引： 

    db.yourColl.ensureIndex({},{background:true});

  14. 批量和精确删除： 

    db.runCommand({dropIndexes:"yourColl",index:"name_-1"}); 精确删除一个
db.runCommand({dropIndexes:"yourColl",index:"*"}); 删除所有

2D索引

平面图中，很多个二维坐标的点。

  1. 查询出距离点（70,180）最近的3个点：

    //添加2D索引
db.yourColl.ensureIndex({"gis":2D},{min:-1,max:201});     //gis属于yourColl
db.yourColl.find({"gis":{$near:[70,180]}}).limit(3);

  2. 查询以点（50,50）和（190,190）为对角线的正方形中的是有点：

    db.yourColl.find({gis:{"$within":[[50,50],[190,190]]}});

  3. 查询以圆心为（56,80）半径为50规则下的圆形面积中的点：

    db.yourColl.find({gis:{$within:{$center:[[56,80],50]}}});

命令执行器

  1. 用命令执行器完成一次删除表的操作：
db.runCommand({drop:"yourColl"})
{
        "nIndexesWas" : 2,
        "msg" : "indexes dropped for collection",
        "ns" : "yourDB.yourColl",
        "ok" : 1
}

  1. 查询mongoDB为我们提供的命令：

    在shell中执行 db.listCommands()
访问网址http://localhost:28017/_commands

  2. 查询服务器版本号和主机操作系统：

    db.runCommand({buildInfo:1})

  3. 查询执行集合的详细信息,大小,空间,索引等：

    db.runCommand({collStats:"yourColl"})

  4. 查看操作本集合最后一次错误信息：

    db.runCommand({getLastError:"yourColl"})

固定集合

  1. 固定特性

    1.1固定集合默认是没有索引的就算是_id也是没有索引的
1.2由于不需分配新的空间他的插入速度是非常快的
1.3固定集合的顺是确定的导致查询速度是非常快的
1.4最适合的是应用就是日志管理

  2. 创建固定集合：

    2.1创建一个新的固定集合要求大小是100个字节,可以存储文档10个
     db.createCollection("yourColl",{size:100,capped:true,max:10})
2.2把一个普通集合转换成固定集合
     db.runCommand({convertToCapped:”yourColl”,size:100000})

  3. 反向排序,默认是插入顺序排序.

    3.1查询固定集合yourColl并且反响排序
     db.yourColl.find().sort({$natural:-1})

  4. 尾部游标,可惜shell不支持java和php等驱动是支持的

    4.1尾部游标概念
     这是个特殊的只能用到固定级和身上的游标,他在没有结果的时候也不回自动销毁他是一直等待结果的到来

关闭服务

  1. Ctrl+C关闭

    如果以前台方式启动 MongoDB 服务，使用“Crtl+C” 服务会关闭，这种关闭方式会等待当前进行中的的操作完成，所以依然是干净的关闭方式。
