Java Applet是用Java语言编写的一些小应用程序，这些程序可以直接内嵌到页面中，由支持Java的浏览器（IE或是Netscape）解释执行能够产生特殊效果的程序。可以大大提高Web页面的交互能力和动态执行能力。包含Applet的网页被称作Java-powered页，可以称其为Java支持的网页。

实例

  1. java类

    import java.applet.Applet;
import java.awt.Graphics;

public class JavaApplet extends Applet{

    public void paint(Graphics g) {
        g.drawString("HelloWorld", 5, 35);
    }

}

  2. Html文件

    <!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Test Applet</title>
</head>
<body>
    <OBJECT   
      classid="clsid:8AD9C840-044E-11D1-B3E9-00805F499D93" 
      width="200" height="200"> 
      <PARAM name="code" value="JavaApplet.class"> 
    </OBJECT> 
</body>
</html>

    Java类不要放在自定义的包中，否则编译之后和Html文件不在同一目录下。

    HTML5 中不支持 &lt;applet> 标签。请使用 object 元素 标签代替；HTML 4.01 中不赞成使用 <applet> 元素。

<object> 标签用于包含对象，比如图像、音频、视频、Java applets、ActiveX、PDF 以及 Flash。

生命周期

共有四种状态（初始态、运行态、停止态和消亡态）和四个方法：init()、start()、stop()和destroy().

  1. init()

    当创建Java Applet且第一次使用支持Java的浏览器载入时，就会执行init()方法。在Java Applet生命周期中，该方法只执行一次，因此可以利用这一点在init()方法中进行一些只需执行一次的初始化操作，例如对变量的初始化等。

  2. start()

    调用完init()方法后，系统将自动调用start()方法。当用户离开包含Applet的主页后又再返回时，或者当浏览器从图标状态恢复为窗口时，系统都会自动再执行一遍start()方法。

    和init()方法不同，start()方法在Applet的生命周期中被多次调用，该方法是Applet的主体。在start()方法中，可以执行一些任务，或者是启动相关的线程来执行任务，如循环播放歌曲等。

  3. stop()

    和star()方法相对应，当用户离开Applet所在页面或者是浏览器变成图标时，系统都会调用stop()方法，因此该方法也是可以被多次调用的。

    stop()方法起到这样的作用：当用户在当前时刻并不十分关注Applet时，停止一些耗用资源的工作，这样就可以提高系统的运行速度，而且系统会自动调用该方法，并不需要人为干预。倘若编写的Applet中不涉及动画等多媒体，一般不必重写该方法。

  4. destroy()

    当用户关闭浏览器时，系统就会调用destroy()方法，应该注意stop()方法和destroy()方法的区别

Java Applet编程显示图像

Java Applet常用来显示存储GIF文件中的图像。

Graphics类的drawImage方法用来显示Image图像。为了提高图像的显示效果，许多Applet都采用双缓冲技术：首先把图像装入内存，然后再显示在屏幕上。

public class JavaApplet extends Applet{
    private Image picture;
    private boolean imageLoaded = false;

    public void init() {
        picture = getImage(this.getCodeBase(), "Penguins.jpg");   //装载图像
        Image offScreenImage = createImage(getSize().width, getSize().height);
        //用方法createImage创建Image对象
        Graphics offScreenGraphics = offScreenImage.getGraphics();
        //获取Graphics对象
        offScreenGraphics.drawImage(picture, 0, 0, this);   
        //当Applet调用drawImage将创建一个调用imageUpdate方法的线程。
        imageLoaded = true;
    }

    public void paint(Graphics g) {
        if(imageLoaded) {
            g.drawImage(picture, 0, 0, null);   
            //显示图像，第四个参数为null。防止再次创建线程
            showStatus("Done");
        } else {
            showStatus("Loading image");
        }
    }

    public boolean imageUpdate(Image img, int infoflags, int x, int y, int w,
            int h) {
        if(infoflags == ALLBITS) {
            imageLoaded = true;
            //infoflags使Applet能够知道图像装入内存的情况。
            repaint();
            return false;
        } else {
            return true;
        }
    }    
}

Java Applet响应鼠标键盘

与用户的交互是Java的主要作用，用户可以通过鼠标与Java Applet程序对话。

public class MouseTest extends Applet{
    String text = "";

    @Override
    public void paint(Graphics g) {
        g.drawString(text, 20, 20);
    }

    @Override
    public boolean mouseDown(Event evt, int x, int y) {
        text = "Mouse Down";
        repaint();
        return true;
    }

    @Override
    public boolean mouseUp(Event evt, int x, int y) {
        text = "Mouse Up";
        repaint();
        return true;
    }

    @Override
    public boolean keyDown(Event evt, int key) {
        text = "key Down";
        repaint();
        return true;
    }

    @Override
    public boolean keyUp(Event evt, int key) {
        text = "Key Up";
        repaint();
        return true;
    }
}
