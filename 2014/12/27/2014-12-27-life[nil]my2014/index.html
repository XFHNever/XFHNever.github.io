我的2014

By Never

发表于 Dec 27 2014

文章目录

感觉时间过得越来越快，转眼就要迎接新的一年了，也是时候对自己过去的一年做个简单的总结了。

##回顾2014

###学习
2014其实感觉在学校基本就没怎么上课，一直在追求着会做项目以及知识的广度上，所以在学校的那段时间基本就是做各种乱七八糟的东西：PHP写blog，帮朋友做些简单的小网站；初涉Grails；了解MongoDB；学习cocos2dx，照着别人的博客做了一个飞机大战….很多时候过于急功近利，反而忘记了根本，学习PHP不去好好学习基本语法，反倒在最开始的时候就去学习框架，尽管使用框架做项目是easy了特别多，但是对PHP语言却基本不甚了解。其他技术也大抵如此。

实习之后，确实越来越感觉基础知识的重要性，尤其是每天上下班给了我充足的时间去看各种书。下面就是本年度读的主要书籍，基本都是实习之后读的：

###技术

  1. 人月神化
  2. 编程之美
  3. 领域特定语言［仍未看完］
  4. 七周七语言
  5. 改变未来的九大算法
  6. 深入理解Java虚拟机：JVM高级特性与最佳实践
  7. Grails Getting Started-II
  8. 高性能Mysql ［仍未看完］
  9. Effictive Java —2014/12/15
  10. 软件管理沉思录－SEI的项目管理、人际沟通和团队协作要诀
  11. 深入浅出nodejs

###非技术

  1. 九州华胥引
  2. 龙族
  3. 一个.很高兴遇见你
  4. 从你的全世界路过
  5. 三体
  6. 全职高手
  7. 乌合之众
  8. 怪诞心理学
  9. 路西法效应 ［仍未看完］

最让我高兴的是：搭建了自己的blog，并一直在更新，将自己的学习历程纪录在上面是一种非常nice的体验（知乎即视感。。），积累的力量是无穷的。

###实习
6月～11月，在eBay实习；11月至今，在天天动听。两个不同的公司，不同的企业文化，从eBay到ttpod，我仍然在尝试不同的公司，去找一个最适合自己发展的环境。

###求职
说多了都是泪：

  1. 找工作还是需要好好提前准备一下的；
  2. 算法很重要啊！！不然连面试机会都没有。。
  3. 还是回南京找工作更好啊。。。和上海不是一个体系的啊。。
  4. 有机会是千万不要错过的。。。懒就可能错误一些蛮好的面试机会。

###生活
一个词：千篇一律。。。。经过了一段时间的低谷，慢慢就又都回到了正轨。生活总还是会继续，没有谁离开不了谁，过眼云烟，慢慢都会看淡。到上海实习，Get了新技能：厨艺。。哈哈哈，现在已经可以脱离菜谱烧菜了！！还更加意识到一个事实：我实在是路痴的不得了！！！

##展望2015
这里希望给自己新的一年定下一些目标，来激励自己不断去学习，不断去进步。

###学习

  1. Blog当然要继续更新了；
  2. 读书是必须的哇，希望2015年里技术书籍10+， 非技术书籍15+；英文书籍2+
  3. 购买VPS，自己建站～（blog或者其他）
  4. 学习算法，希望可以做完leetcode并且阅读完算法大全；
  5. 至少做一个高质量的上线app；
  6. 阅读项目源码1+；
  7. 继续活跃在segmentfault；

###实习
当然是继续好好实习，尽量学习更多技术了。

###毕业

  1. 做完毕设，顺利毕业；
  2. 有一场难忘的毕业旅行；
  3. 在最后的两个月里和好朋友们好好玩。

###生活
对自己要求很低的。。天天开心！！其他事皆随缘～
