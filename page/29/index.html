分片简介

分片是指将数据拆分,将其分散存在不同机器上的过程.有时也叫分区.将数据分散在不同的机器上,不需要功能强大的大型计算机就可以存储更多的数据,处理更大的负载.

MongoDB支持自动分片,可以摆脱手动分片的管理.集群自动切分数据,做负载均衡.

By Never

发表于 Mar 2 2014

基本概念

副本集就是有自动故障恢复功能得主从集群。

主从集群和副本集最大的区别就是副本集没有固定的“主节点”；整个集群会选出一个“主节点”，当其挂掉后，又在剩下的从节点中选中其他节点为“主节点”，副本集总有一个活跃点(primary)和一个或多个备份节点(secondary)。

Mongodb（一）--Win7下安装配置

By Never

发表于 Feb 28 2014

主要步骤

  1. 从MongoDB downloads page下载对应系统的最新版本MongoDB。
  2. 解压安装到任意目录，我的在 D:\Program Files\mongodb。

  3. 在mongodb目录下建立data文件夹，然后在data文件下创建db和log文件夹，然后在log文件夹下建立mongodb.log。

    md data  
md data\db  
md data\log 
copy con data\log\mongodb.log 然后回车，输入想要输入的内容，然后回车按Ctrl+Z，（或者直接到文件夹下手动创建）

  4. 启动服务：运行cmd.exe 进入DOS界面

    >cd D:\Program Files\mongodb\bin
>D:\Program Files\mongodb\bin> mongod -dbpath "D:\Program Files\mongodb\data\db"

    双击bin下mongod.exe启动程序，再运行mongo.exe。
